# SaintsLND Secure Checkout
This project has been modernized to use Vite, Tailwind CSS, and Firebase.

## Tech Stack
- **Vite**: Fast development server and build tool.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Firebase**: Backend for authentication and database (Firestore).
- **Stripe**: Payment processing integration.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Start the backend server (for Stripe):
   ```bash
   npm run server
   ```

## Project Structure
- `src/`: Contains source code (JS, CSS).
- `index.html`: Main entry point.
- `server.js`: Express backend for Stripe.
- `tailwind.config.js`: Tailwind configuration.
- `vite.config.js`: Vite configuration.
